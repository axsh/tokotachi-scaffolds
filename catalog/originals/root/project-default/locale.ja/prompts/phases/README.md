# フェーズ

このディレクトリは、プロジェクトのフェーズごとに仕様書と計画を管理します。

## 構成

```
phases/
  000-foundation/      # フェーズ 0: 基盤構築
    branches/
      000-branch-name/ # フィーチャーブランチ名
        ideas/         # 仕様書
        plans/         # 実装計画
    refs/              # このフェーズの参考資料
  001-<next-phase>/    # フェーズ 1: ...
    branches/
      000-branch-name/ # フィーチャーブランチ名
        ideas/         # 仕様書
        plans/         # 実装計画
    refs/              # このフェーズの参考資料
```

## ワークフロー

1. `<ブランチ名>/ideas/` に仕様書を作成
2. 仕様書をレビュー・承認
3. `<ブランチ名>/plans/` に実装計画を作成
4. 実装計画をレビュー・承認
5. 計画に基づいて実装を実行
